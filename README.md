# 🌍 LexLang – Langue Mina (Gen, Gɛ̃)

Projet numérique collaboratif visant à constituer une **base lexicale publique structurée** pour la langue **Mina** (ou Gen), dans le cadre du programme *LexLang*. L'objectif est de documenter, diffuser et rendre interopérable cette ressource linguistique pour des usages en **éducation**, **technologie linguistique**, **intelligence artificielle** et **préservation du patrimoine culturel**.

---

## 🧠 À propos de la langue Mina

- **Nom local :** Gɛ̃
- **Noms alternatifs :** Gen, Guen
- **Famille linguistique :** Niger-Congo → Kwa → Gbe
- **Code ISO 639-3 :** `gej`
- **Pays :** Togo (région Maritime), Ghana (Volta Region)
- **Statut :** Langue véhiculaire, très proche de l’éwé

---

## 📦 Fichier principal : `lexique.csv`

Le fichier contient un **lexique structuré** avec les champs suivants :

| Champ           | Description |
|----------------|-------------|
| `mot`           | Mot ou expression en mina |
| `prononciation` | Transcription phonétique (format IPA ou usage local) |
| `catégorie`     | Nature grammaticale (nom, verbe, adjectif, etc.) |
| `traduction_fr` | Traduction en français |
| `exemple`       | Phrase d’exemple avec traduction |
| `variante`      | Variante dialectale si existante |

### Exemple de ligne :

```csv
mot,prononciation,catégorie,traduction_fr,exemple,variante
afɔ,àfɔ́,nom,pied,"Me nɔ́ afɔ gake e wò dze.", 
```

---

## 🎯 Objectifs du projet

- Documenter un lexique de base de 1 000 mots courants
- Structurer les données pour usage en IA (tokenisation, TTS, NLP)
- Fournir une ressource libre pour les enseignants et locuteurs
- Créer un pont entre langue minoritaire et technologies numériques
